Fluxo do Processador de Registros para o padrão de Record Distribution para WebSphere MQ: Unidirecional

Detalhes sobre o fluxo de RecordProcessor para o padrão de Record Distribution para WebSphere MQ: Unidirecional.

Este subfluxo não possui, por padrão, nenhum impacto. Quando o padrão de Record Distribution para WebSphere MQ: Unidirecional é usado para gerar uma instância padrão, o fluxo de RecordProcessor contém apenas um nó de Passagem. Este subfluxo é incluído para fornecer um local onde uma instância padrão pode ser customizada usando o conjunto de ferramentas padrão, sem alterar a estrutura do padrão.
