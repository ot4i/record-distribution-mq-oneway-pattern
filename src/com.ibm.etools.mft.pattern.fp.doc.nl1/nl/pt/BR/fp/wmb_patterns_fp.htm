Processamento de Arquivos

Um barramento de serviço corporativo (ESB) fornece um ambiente de execução gerenciado para o processamento de arquivos locais que são copiados usando um protocolo FTP.

Os padrões de Processamento de Arquivo fornecem suporte para atividades como:
  • Transformação e conversão de dados que são mantidos nos arquivos
  • Retalhação de arquivos em múltiplos registros de transação individuais
  • Roteamento de registros
  • Acumulação de registros em arquivos de destino
  • Roteamento de arquivos inteiros para locais e processadores especificados
  • Roteamento de registros em arquivos para locais e processadores especificados

Estes padrões também suportam a inclusão de utilitários de gerenciamento padrão, por exemplo, segurança, criação de log, monitoramento e cobrança.
