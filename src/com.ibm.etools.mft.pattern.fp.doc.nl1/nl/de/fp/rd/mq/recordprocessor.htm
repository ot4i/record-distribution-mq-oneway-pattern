Datensatzprozessor-Nachrichtenfluss für das Muster Record Distribution to WebSphere MQ: one-way (Verteilung von Datensätzen an WebSphere MQ: unidirektional)

Informationen zum Datensatzprozessor-Nachrichtenfluss für das Muster Record Distribution to WebSphere MQ: one-way (Verteilung von Datensätzen an WebSphere MQ: unidirektional).

Dieser untergeordnete Nachrichtenfluss hat standardmäßig keine Auswirkungen. Wenn das Muster 'Record Distribution to WebSphere MQ: one-way' (Verteilung von Datensätzen an WebSphere MQ: unidirektional) für die Generierung einer Musterinstanz verwendet wird, enthält der Datensatzprozessor-Nachrichtenfluss nur einen Pass-through-Knoten. Dieser untergeordnete Nachrichtenfluss wird eingefügt, um einen Bereich bereitzustellen, in dem einer Musterinstanz mithilfe der Standardtools angepasst werden kann, ohne dass die Struktur des Musters geändert wird.
