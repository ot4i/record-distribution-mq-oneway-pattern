Tâches à exécuter avant d'appliquer le modèle Distribution d'enregistrements vers WebSphere MQ : unidirectionnel

Avant d'appliquer le modèle pour produire une instance de modèle, vous devez déterminer les tâches à exécuter parmi les tâches suivantes. Si nécessaire, exécutez les tâches suivantes :

  • Vérifiez que le modèle répond aux besoins de l'entreprise et qu'il peut être pris en charge par les créateurs de fichiers et les consommateurs de messages.
  • Affectez un nom à l'instance de modèle. Identifiez la fonction de l'application de modèle pour déterminer le nom approprié.
  • Déterminez si vous devez utiliser la consignation. Si tel est le cas, vous devez déterminer le gestionnaire de files d'attente et la file d'attente auxquels les messages de consignation sont envoyés. La consignation doit être exécutée dans une file d'attente de consignation générale qui existe déjà pour d'autres flux de courtier ou dans une file d'attente créée pour le modèle d'instance.
  • Déterminez si vous avez besoin de générer des messages d'erreur. Si tel est le cas, vous devez déterminer le gestionnaire de files d'attente et la file d'attente auxquels les messages d'erreur doivent être envoyés.
  • Déterminez le mode de détection des enregistrements et, si nécessaire, préparez un ensemble de messages pour définir le mode d'analyse des enregistrements.
  • Déterminez si la validation est nécessaire. Si tel est le cas, préparez un ensemble de messages pour définir le mode d'analyse des enregistrements.
  • Déterminez si les fichiers non traités sont locaux ou distants. Si des fichiers distants doivent être traités, le nom de service configurable doit être identifié.
  • Vérifiez si plusieurs destinations possibles existent. Si tel est le cas, sélectionnez la méthode de routage à utiliser.
    • Si vous disposez d'un nombre limité de routes dans lesquelles les destinations ne changent pas souvent, affectez au paramètre Acheminement la valeur Indiquer les routes.
    • Si le groupe de routes cible n'est pas complètement déterminé ou peut changer fréquemment, affectez au paramètre Acheminement la valeur Rechercher les routes. Si vous sélectionnez cette valeur, le nom de la source de données doit être connu.
