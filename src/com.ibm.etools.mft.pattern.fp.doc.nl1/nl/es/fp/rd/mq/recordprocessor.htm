Flujo RecordProcessor para el patrón Distribución de registros en WebSphere MQ: unidireccional

Detalles sobre el flujo RecordProcessor para el patrón Distribución de registros en WebSphere MQ: unidireccional.

De forma predeterminada, este subflujo no tiene ningún efecto. Cuando se utiliza el patrón Distribución de registros en WebSphere MQ: unidireccional para generar una instancia de patrón, el flujo RecordProcessor sólo contiene un nodo Passthrough. Este subflujo se incluye para proporcionar un lugar donde una instancia de patrón se puede personalizar utilizando herramientas estándar, sin cambiar la estructura del patrón.
