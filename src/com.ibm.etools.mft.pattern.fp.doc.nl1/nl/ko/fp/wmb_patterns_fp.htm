파일 처리

Enterprise Service Bus(ESB)는 FTP 프로토콜을 사용하여 복사한 로컬 파일 처리를 위해 관리되는 프로덕션 환경을 제공합니다.

파일 처리 패턴은 다음과 같은 활동을 지원합니다.
  • 파일에 보관된 데이터 변환
  • 여러 개의 개별 트랜잭션 레코드로 파일 나누기
  • 레코드 라우팅
  • 대상 파일로 레코드 축적
  • 지정된 위치 및 프로세서로 전체 파일 라우팅
  • 지정된 위치 및 프로세서로 파일 안의 레코드 라우팅

이 패턴은 보안, 로그 기록, 모니터링 및 충전 같은 표준 관리 유틸리티도 포함하도록 지원합니다.
