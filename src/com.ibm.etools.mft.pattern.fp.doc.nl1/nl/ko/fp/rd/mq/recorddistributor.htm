WebSphere MQ에 대한 레코드 분배: 단방향 패턴의 레코드 분배자 플로우

WebSphere MQ에 대한 레코드 분배: 단방향 패턴의 레코드 분배자 플로우 세부사항입니다.

플로우는 다음과 같은 조치를 수행합니다.

  1. 파일 입력 매개변수에 따라 처리할 파일을 선택합니다.
  2. 레코드 감지 매개변수에 따라 파일에서 레코드를 읽습니다. 각 레코드는 트랜잭션 안에서 처리됩니다. 유효성 검증이 지정되면 메시지 세트에 대해 레코드의 유효성을 검증합니다.
  3. 라우트 서브플로우를 호출하여 올바른 WebSphere MQ 목적지를 판별하고 레코드를 올바른 대상 큐에 기록합니다. 기본적으로 파일에서 읽을 때와 동일한 형식의 레코드로 큐에 기록합니다. 필요하면, 레코드 프로세서 플로우를 사용자 정의하여 레코드를 다른 형식으로 변환할 수 있습니다.
  4. 라우팅하지 않음이 선택되면 레코드가 기본 메시지 큐에 기록됩니다.
  5. 오류가 발생하면(예: 레코드 읽기 또는 레코드를 큐에 기록 실패), 오류 메시지가 기록되고 플로우가 백아웃됩니다. 파일이 백아웃 디렉토리에 기록됩니다.
  6. 모든 데이터를 처리하면 로그 서브플로우를 호출하여 완료를 기록합니다. 레코드가 기본 메시지 큐에 기록된 경우 이 조치에 주의합니다.
  7. 플로우가 성공적으로 완료되고 파일이 아카이브 서브디렉토리로 이동합니다.
